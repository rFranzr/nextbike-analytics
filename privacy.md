# Privacy Policy

**Last Updated:** December 2025

## Overview

Nextbike Analytics is committed to protecting your privacy. This application is designed with privacy as a core principle, operating entirely client-side without any backend infrastructure.

## Data Collection and Storage

### No Cookies
This application **does not set or use any cookies**. Your browsing session is not tracked through cookie-based mechanisms.

### No Backend Server
This application operates **100% client-side** with no backend server. All processing happens directly in your browser.

### Client-Side Data Storage
- **Session Storage**: Your login credentials and ride data are temporarily stored in your browser's session storage only
- **Automatic Cleanup**: Session data is automatically cleared when you close your browser tab or log out
- **No Persistent Storage**: No data is stored permanently on your device or any server

## Third-Party Services

### Nextbike API
- Your credentials are sent **directly** to the official Nextbike API
- We do not intercept, store, or process your credentials
- All API communication happens directly between your browser and Nextbike's servers
- Please refer to [Nextbike's privacy policy](https://www.nextbike.net/en/privacy/) for information about how they handle your data

### GitHub Pages Hosting
This website is hosted on **GitHub Pages**. GitHub may collect and log certain metadata when you access this site, including:
- IP addresses
- Browser type and version
- Access times
- Referrer information

This metadata collection is standard for web hosting services and is governed by [GitHub's Privacy Statement](https://docs.github.com/en/site-policy/privacy-policies/github-privacy-statement).

We do not have access to or control over this metadata. GitHub uses this information for security, analytics, and service improvement purposes.

## Data Transmission

- **No Third-Party Data Transmission**: Your ride data is never transmitted to any third-party service other than the official Nextbike API
- **Direct API Access**: All data requests go directly from your browser to Nextbike's servers
- **No Data Interception**: We do not intercept, copy, or store any of your ride data

## Your Rights

You have full control over your data:

- **Clear Data Anytime**: You can clear your session data at any time by logging out
- **No Account Required**: You are not required to create an account with us
- **Browser Control**: You can clear your browser's session storage at any time through your browser settings

## Analytics

This website uses Umami Analytics, a privacy-focused analytics service. Umami:
- Does not use cookies
- Does not collect personal data
- Anonymizes IP addresses
- Complies with GDPR, CCPA, and PECR

For more information, please visit [Umami's privacy policy](https://umami.is/privacy).

## Changes to This Privacy Policy

We may update this Privacy Policy from time to time. Any changes will be posted on this page with an updated "Last Updated" date.

## Contact

If you have any questions about this Privacy Policy, please open an issue on our [GitHub repository](https://github.com/rfranzr/nextbike-analytics).
